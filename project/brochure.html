How does it work?

First, select any number of countries from the country list provided. Next, select any number of facts from the fact list (forest area percent is already checked for you). Click submit and you are presented with tables of data for each country you have selected.
Does forest area percent correlate to any of these other facts?

Who is it for?

This app is for any curious humans wondering about how big forests are, and if this size correlates to any other facts. It can even inspire them to travel to these places!
